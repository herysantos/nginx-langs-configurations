# NGINX LANGS CONFIGURATIONS

This repository has code references to configure an Nginx web server with docker to work with different programming languages.

Open an [Issue](https://github.com/herysantos/nginx-langs-configurations/issues) if you notice anything that can be improved or is not a good practice.

I hope this repo to be useful to you.

All docs and commands are based on a Ubuntu OS environment. If you are using another OS or distributions, open an [Issue](https://github.com/herysantos/nginx-langs-configurations/issues) asking your doubts.
